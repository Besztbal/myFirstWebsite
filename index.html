biológus

"Kutatóként tanulmányozom a sejtekben lejátszódó komplex folyamatokat, amihez mikroszkópot használok. Ez magában foglalja a minták előkészítését, vizsgálatukat és felvételek készítését mikroszkóp alatt. A kapott képeket feldolgozom, kiértékelem."



Definíciók:

Logikai kapu:Egy olyan idealizált vagy fizikai eszköz ami logikai alapmüveletet valósít meg egy vagy több bináris bemeneten és produkál egy bináris kimenetet.

ASCII:Az angol neve American Standard Code for Information Interchange. A számítógépek csak a számokat értik, így a 'a' vagy '@' karakterek értelmezéséhez szükséges számosított értékeket írja le.

RAM (Ramdom Access Memory):Egy véletlen elérésű írható–olvasható adattároló eszköz. Tárolja a CPU által végrehajtandó programokat és a feldolgozásra váró adatokat. Egy adott memóriarész elérésének gyorsasága független az elhelyezkedésétől.

Programozási nyelv:A számítástechnikában használt olyan, az ember által olvasható és értelmezhető utasítások sorozata, amivel közvetlenül, vagy közvetve (például: gépi kódra fordítás után) közölhetjük a számítógéppel egy adott feladat elvégzésének módját.

Domain Name System (tartománynévrendszer):Egy hierarchikus, nagymértékben decentralizált elnevezési rendszer számítógépek, szolgáltatások, illetve az internetre vagy egy magánhálózatra kötött bármilyen erőforrás számára.

Internet:Egy alkalmazás protokol elosztott, együttműködő és hipermédiás információs rendszerek részére. A World Wide Web adat kommunikációjának alapja. Ez az alapvető protokol a hypertext cseréjére és mozgatására.
